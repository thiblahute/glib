# Description

`gtester-report` is a script which converts the XML output generated by
gtester into HTML.

# Options

  - `-h`, `--help`  
    print help and exit

  - `-v`, `--version`  
    print version information and exit

  - `-s`, `--subunit`  
    Output subunit. Needs python-subunit.
